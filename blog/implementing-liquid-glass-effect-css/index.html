Implementing a Liquid Glass UI Effect with CSS

Abbas Uddin
25/10/2025 7 min read

The "liquid glass" or glassmorphism effect is a popular UI trend that adds a frosted, semi-transparent look to elements, making them feel layered and modern. It relies heavily on CSS properties like backdrop-filter and subtle shadows. Let's walk through how to integrate this effect into an existing project.

Step 1: Core CSS Setup (Variables & Base Styles)

First, define the core colors and styles in your main CSS file (e.g., global.css or style.css). Using CSS variables makes customization easier later. 

                        

                            

                            
                        

                        /* Add or merge these CSS Variables */
:root {
  --c-glass: #bbbbbc; /* Base color for glass */
  --c-light: #fff;    /* Light color for highlights/borders */
  /* ... rest of variables ... */
  --saturation: 150%;   /* Saturation for backdrop filter */

  /* Keep your existing project colors here too! */
  --primary-color: #2c3e50;
  /* ... other project variables */
}

/* Add or modify these body styles */
body.liquid-glass-theme {
    font-family: 'Inter', sans-serif; /* Requires importing 'Inter' font */
    /* ... rest of body styles ... */
    overflow-x: hidden; /* Prevent horizontal scroll */
}

Note: You'll need to import the 'Inter' font (see Step 4) for the specified font-family to work. Decide whether to apply the body styles globally or conditionally using a class like .liquid-glass-theme.

Step 2: Define the Glass Effect Class

Create a reusable CSS class... (text as before) 

                        

                            

                             
                        

                        /* New glass effect class */
.glass-effect {
    background-color: color-mix(in srgb, var(--c-glass) 12%, transparent);
    backdrop-filter: blur(12px) saturate(var(--saturation));
    -webkit-backdrop-filter: blur(12px) saturate(var(--saturation)); /* Safari support */
    border-radius: 1.5rem; /* Default rounding */
    /* ... rest of box-shadow styles ... */
}

Step 3: Add Animated Background Shapes

To enhance the effect... (text as before)

First, add the HTML structure...

                        

                            

                             
                        

                        <body class="liquid-glass-theme"> <!-- Optional theme class -->
    <div class="background-shapes">
        <div class="shape shape1"></div>
        <div class="shape shape2"></div>
        <div class="shape shape3"></div>
    </div>
    <!-- Rest of your page content -->
</body>

Then, add the corresponding CSS:

                        

                            

                             
                        

                        /* Styling for the background gradient shapes */
.background-shapes { /* ... styles as before ... */ }
.shape { /* ... styles as before ... */ }
.shape1 { /* ... styles as before ... */ }
.shape2 { /* ... styles as before ... */ }
.shape3 { /* ... styles as before ... */ }
@keyframes animateShape { /* ... keyframes as before ... */ }

Step 4: Import Google Font

Add the 'Inter' font link... (text as before) 

                         

                            

                             
                        

                        <head>
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
    <!-- Your other stylesheets -->
</head>

Step 5: Apply the Glass Effect & Adjust Styles

Now, add the .glass-effect class... (text as before) 

                         

                            

                             
                        

                        <!-- Example: Applying to a header -->
<header class="glass-effect">
    <!-- Header content -->
</header>

<!-- Example: Applying to a product card -->
<div class="product-card glass-effect">
    <!-- Card content -->
</div>
